# Interactive Sales Dashboard

## Overview

This project is an Interactive Sales Dashboard developed using Python, Streamlit, and Plotly.

The dashboard helps visualize sales data through interactive charts and KPIs.

## Features

- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Top Selling Products
- Interactive Filters
- Download Filtered Data

## Technologies

- Python
- Streamlit
- Pandas
- Plotly

## Dataset

Use the Sample Superstore Dataset from Kaggle.

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Interactive-Sales-Dashboard.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

## Output

Interactive dashboard with real-time filtering and visual analytics.
